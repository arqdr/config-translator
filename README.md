# Конфигурационный транслятор

## Общее описание
Инструмент командной строки для преобразования текста из учебного конфигурационного языка в TOML формат. Поддерживает словари, константы, строки, числа и константные вычисления в постфиксной форме.

## Описание всех функций и настроек
### Синтаксис входного языка:
- Однострочные комментарии: `% Это комментарий`
- Числа: `[+-]?\d+`
- Словари: `@{ имя = значение; ... }`
- Имена: `[_a-z]+`
- Значения: числа, строки, словари
- Строки: `[[Это строка]]`
- Константы: `имя = значение`
- Константные выражения: `$имя 1 +$`

### Операции константных выражений:
1. Сложение
2. Вычитание  
3. Умножение
4. Деление
5. mod() - функция взятия остатка

### Ключи командной строки:
- `-o, --output` - путь к выходному TOML файлу (обязательный)
- `-h, --help` - справка по использованию
